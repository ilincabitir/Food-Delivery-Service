# Java Food Delivery CLI
A lightweight, terminal-based food delivery simulation built with Java. This project demonstrates core OOP principles, file handling for data persistence, and a command-line interface for user interaction.

# Installation & Running

## Clone the repository
```
git clone https://github.com/ilincabitir/Food-Delivery-Service
```
## Navigate to the directory:
```
cd food-delivery-java
```
## Compile the application:

```
javac src/*.java -d bin
```
## Run the app:
```
java -cp bin Main
```
